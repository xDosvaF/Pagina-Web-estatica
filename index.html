Interfaces de Programación de Aplicaciones (APIs)

Las APIs permiten que diferentes sistemas se comuniquen entre sí de manera estandarizada.

Tipos de APIs

TipoCaracterísticas
RESTBasado en HTTP, ligero y muy popular
SOAPMás complejo, basado en XML
GraphQLConsultas flexibles y eficientes

Bases de Datos

Las bases de datos almacenan y organizan la información para que pueda ser consultada y modificada eficientemente.

Clasificación

  1. Relacionales (SQL) — estructura tabular con esquema fijo
  2. No Relacionales (NoSQL) — flexibles, orientadas a documentos, grafos o clave-valor

Mecanismos de Comunicación

Los sistemas utilizan protocolos y APIs para intercambiar datos de manera segura y eficiente.

Protocolos comunes

  • HTTP/HTTPS — transferencia de hipertexto
  • WebSockets — comunicación bidireccional en tiempo real
  • MQTT — mensajería ligera para IoT

Seguridad en la Comunicación

La seguridad es fundamental en el intercambio de datos entre APIs y bases de datos.

Buenas prácticas

  1. Uso de HTTPS en todas las comunicaciones
  2. Autenticación y autorización (OAuth 2.0, JWT)
  3. Cifrado de datos en reposo y en tránsito
